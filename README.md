## Como Executar o Projeto

<br/>

###  Requisitos:

* Maven instalado;
* Nenhum serviço rodando na porta 8080;

### 1 - Entre no diretório ./doeaqui do projeto;
<br/>

### 2 - Abra o terminal e execute o comando abaixo:

```console

```
